
## （1）制作アプリのタイトル
認知症コミュニケーションチャットボット「だいちゃん」

## （2）制作アプリの説明（40文字程度）
認知症高齢者と最適なチャットができる
※スマホかタブレットかPCが必要なので、実質使えないアプリ💦

## （3）工夫した点・こだわった点
- 自社プロダクトの認知症コミュニケーションロボット「だいちゃん」の会話ロジックをできるだけ再現した
- UI/UXにこだわり、使い易いが、LINEのようなUIではなく、Chat-GPTやGeminiのようなモダンなUIにした
- いずれ会話データから認知症の傾向や進行具合を判断できるようにしたいので、年月日毎に会話データを保存し、過去の会話は編集できないようにした 

## （4）難しかった点・次回トライしたいこと（または機能）
- アホな会話、脈略のない会話にならないが、認知症高齢者の方に好まされそうな会話にチューニングするのが大変だった
- APIを無料で使っているので、無料枠が終わるとAPIエラーが出てしまい、そのエラーの原因に気付かず時間がかかり過ぎた
- 会話データから認知症の傾向や進行具合を判断できるようにしたい
- チャットボットではなく、音声対話でやり取りできるようにしたい
- 最後の最後にAPIキーを公開せずにGitHubに上げてPagesで公開するのに大苦戦した
